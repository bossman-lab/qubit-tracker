# Qubit Tracker 🧪🔬

Tracking the global race for fault-tolerant quantum computing: architectures, breakthroughs, national strategies, and key milestones.

## What This Is

A living document repository following the most consequential technology race of the next decade. The goal is not to predict winners — it's to track the concrete engineering signals that will tell us who's ahead.

## Architecture Landscape

| Architecture | Key Player | Advantage | Challenge |
|-------------|-----------|-----------|-----------|
| Superconducting transmon | Google, IBM, USTC (China) | Mature, highest gate fidelities | Wiring density at scale |
| Cat qubit (bosonic) | Alice & Bob (France) | Noise-biased, lower QEC overhead | Cavity manufacturing |
| Neutral atom | Pasqal (France) | Flexible connectivity, logical > physical | Gate speed (μs vs ns) |
| Silicon spin | Quobly (France), Intel | Reuses semiconductor fab | Spin coherence |
| Photonic | Quandela (France), USTC (China) | Room-temp operation, HPC integration | Photon loss, two-qubit gates |
| Carbon nanotube | C12 (France) | Novel platform | Earliest stage |
